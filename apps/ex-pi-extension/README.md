# Ex-Plannotator for Pi

Ex-Plannotator provides `/ex-plannotator-last`, a persistent browser surface for reviewing recent assistant responses without changing Official Plannotator.

Build from the repository root:

```sh
bun run build:ex-pi
```

Then load or install `apps/ex-pi-extension` as a Pi package. Official `@plannotator/pi-extension` can remain installed.
